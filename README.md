# nonebot-plugin-cube
Rubik's Cube plugin based on nonebot framework,基于nonebot框架的魔方插件

## 用法
将nonebot_plugin_cube拷贝到bot根目录到src/plugin目录下
在bot.py中添加如下命令
`nonebot.load_plugins('src/plugins/nonebot_plugin_cube')`

## 命令
|  方法  | 命令  |
|  ----  | ----  |
| 开始游戏  | c\|cube\|魔方 |
| 结束游戏  | c结束      |
| 撤销操作  | cbk       |
| 显示排名  | rank\|排名 |
